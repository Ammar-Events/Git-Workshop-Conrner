- **VSCode**  
  A lightweight yet powerful source-code editor with extensions, debugging, Git integration, and customizable features.

- **Notion**  
  An all-in-one workspace used for note-taking, knowledge bases, project management, and collaborative documentation.

- **Figma**  
  A cloud-based design and prototyping tool for UI/UX teams, enabling real-time collaboration and component-based workflows.

- **WSL (Windows Subsystem for Linux)**  
  A compatibility layer that allows you to run Linux distributions and tools directly on Windows without a virtual machine.

- **GitKraken**  
  A user-friendly Git client with visual commit graphs, branch management, merge tools, and GitHub/GitLab integration.
